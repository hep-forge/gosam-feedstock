Feedstock for [gosam](https://github.com/gudrunhe/gosam) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

GoSam: automated one-loop matrix element generation for BSM/SM amplitudes

## Architectures

State per architecture: see the platforms badge above or [anaconda.org/hep-forge/gosam](https://anaconda.org/hep-forge/gosam).


## Install

```bash
conda install -c hep-forge -c conda-forge gosam
```
